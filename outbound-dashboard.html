Connect Google Ads

Add 5 environment variables to Railway and live campaign data will appear here automatically.

  1. Get your Developer Token In Google Ads → Tools & Settings → API Center → Apply for basic access. Approved instantly for test accounts.
  2. Create OAuth2 credentials Go to console.cloud.google.com → APIs & Services → Credentials → Create OAuth 2.0 Client ID (Web app). Add https://developers.google.com/oauthplayground as a redirect URI.
  3. Get a Refresh Token Go to developers.google.com/oauthplayground → ⚙ Settings → Use your own credentials → Scope: https://www.googleapis.com/auth/adwords → Authorize → Exchange for tokens. Copy the Refresh Token.
  4. Add to Railway Variables In your Railway project → Variables, add: GOOGLE_ADS_DEVELOPER_TOKEN, GOOGLE_ADS_CLIENT_ID, GOOGLE_ADS_CLIENT_SECRET, GOOGLE_ADS_REFRESH_TOKEN, GOOGLE_ADS_CUSTOMER_ID (your 10-digit ID, no dashes).
